Thumbnails mode


$('.thumbs').slick({
    thumbs: true,
    adaptiveHeight: true
});

Features

  • Simple initialization, "data-thumb" params in slide
  • Support dynamic changes of slider Add | Remove
  • Auto hide thumbnail arrows where not needed
  • Render thumbnails to outer container.
  • Thumbnails in track auto centered. Useful where not a lot slides
  • Minimal work with DOM. No unnecessary insertions deletions

Thumbs centered

If the image is not enough to fill the scrolling area, the thumbs will be centered.


$('.thumbs--few').slick({
    thumbs: true,
    thumbFrame: false,
    adaptiveHeight: true
});
With 
thumbFrame: false
use 
.slick-thumb.slick-active
to select active slide

Thumbs to outer container


$('.thumbs--container').slick({
    thumbs: true,
    thumbArrows: false,
    appendThumbs: $('.thumbs__container'),
    adaptiveHeight: true
});
